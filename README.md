# Gem–Tutor BI (prompt de sistema do protótipo)

Este repositório reúne o **prompt de sistema** utilizado na configuração do protótipo Gem–Tutor BI, um tutor conversacional com IA generativa concebido para apoiar o estudo da disciplina de Business Intelligence em ambiente sandbox do Google Classroom.

O protótipo foi desenvolvido como parte de um Trabalho de Conclusão de Curso (TCC) em Sistemas de Informação – Faculdade Impacta, focado em personalização leve do estudo em Educação a Distância (EaD), ancorada ao catálogo do AVA/LMS.

---

## 1. Objetivo do repositório

O objetivo deste repositório é:

- disponibilizar o **texto integral do prompt de sistema** utilizado no Google AI Studio;
- permitir **rastreabilidade e reprodutibilidade** do estudo descrito no TCC;
- servir como base para eventuais evoluções futuras (por exemplo, inclusão de scripts de integração).
